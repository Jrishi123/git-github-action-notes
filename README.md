# 🚀 Git & GitHub Actions DevOps Project

## 📌 Overview
This project demonstrates Git, GitHub Actions, Docker, and CI/CD pipeline.

## 🧰 Tools Used
- Git
- GitHub Actions
- Docker
- Node.js

## ⚙️ Workflow
1. Code pushed to GitHub
2. CI pipeline runs (Node build)
3. Docker image is built
4. Image pushed to DockerHub

## 🎯 Goal
To demonstrate end-to-end CI/CD automation.
